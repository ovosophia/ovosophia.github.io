hexo build blog on MAC

1.
install brew
/usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”
注册github账号
brew install git #Mac电脑使用brew安装
创建ssh
$cd ～/.ssh //检查是否已经存在ssh
ssh-keygen -t rsa -C xxxxx@hotmail.com（注册github时的email）
ssh key默认保存地址：生成在 ～/.ssh下
生成后可以在命令行中用open ～/.ssh打开文件夹，打开id_rsa.pub
复制其中的内容到github SSH key管理中
github网站，右上角头像——setting——SSH keys——add and name new key

验证MAC和Github是否连上，ssh -T git@github.com
注意当前路径
关于版本控制：http://www.cnblogs.com/angeldevil/p/3238470.html

git remote add origin git@github.com:youname/Test.git //连接远程仓库并建了一个名叫：origin的别名
git push -u origin master
如果这步出错了：
git pull origin master

brew install node #如果我没记错的话,最新版的node.js的包中已经集成了npm包管理工具

使用以下命令验证是否安装成功
node -v
npm -v

进入存放blog的文件夹
npm install hexo -g #-g表示全局安装, npm默认为当前项目安装
hexo init #执行init命令初始化hexo到你指定的目录
hexo generate #自动根据当前目录下文件,生成静态网页

添加sublime subl到命令行
alias subl=\’’/Applications/Sublime Text 2.app/Contents/SharedSupport/bin/subl’\’
